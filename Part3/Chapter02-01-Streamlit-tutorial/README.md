# Chapter02-01-Streamlit-tutorial

## 준비
```bash
pip install -r requirements.txt
```

## 실행
- 멀티페이지(페이지 목록 포함): 원하는 진입 파일을 실행하면 `pages` 폴더의 페이지들이 좌측 상단에 나타납니다.
```bash
streamlit run app.py        # 기본 인사 예제
streamlit run tutorial.py   # 컴포넌트 데모
```
- 단일 페이지로 별도 확인:
```bash
streamlit run page4.py
```

## 종료
- 실행을 멈출 때는 터미널에서 `Ctrl + C`를 누릅니다.
